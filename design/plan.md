## 6. 验收标准

### 功能验收

- [x] 能打开/创建/保存 Markdown 文件
- [x] 单栏编辑模式，输入语法实时渲染
- [x] 光标位置显示源码，离开后渲染
- [x] 支持所有基础 Markdown 语法
- [x] Ctrl+V 粘贴图片到 img/ 目录
- [x] 目录树浏览和文件操作
- [x] Tab 标签页多文档切换
- [x] 主题切换（亮/暗/跟随系统）
- [x] 自动保存功能
- [x] 快捷键支持

### 性能验收

- [ ] 打开10MB文档 < 3秒
- [ ] 输入响应 < 50ms
- [ ] 图片粘贴 < 1秒
- [ ] 目录树加载 < 500ms
- [ ] 内存占用 < 200MB

### 兼容性验收

- [ ] Windows 10/11
- [ ] 支持 UTF-8 编码
- [ ] 支持长文件名/路径
- [ ] 支持中文路径
